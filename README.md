# HMIS Prototype

**Health Management Information System (HMIS) Prototype** is a web-based application built with **HTML, CSS, and Flask**.  
It allows basic management of patient records and health data, serving as a starting point for a full-featured HMIS.

---

## Features
- Add, edit, view, and delete patient records
- Track basic health metrics (vitals, lab reports, vaccinations)
- Simple search and filter functionality
- Basic reporting (CSV export)

> ⚠️ This is a prototype. Features are limited and may be under development.

---

## Technologies Used
- **Frontend:** HTML, CSS
- **Backend:** Python Flask
- **Database:** SQLite 
- **Other Tools:** Git, GitHub

---

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/3RA31M/hmis-prototype.git
    ```
2. Navigate to the project folder:
    ```bash
    cd hmis-prototype
    ```
3. Create and activate a virtual environment (recommended):
    ```bash
    python -m venv venv
    source venv/bin/activate      # Linux/macOS
    venv\Scripts\activate         # Windows
    ```
4. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
5. Run the Flask app:
    ```bash
    python app.py
    ```
6. Open your browser at [http://127.0.0.1:5000](http://127.0.0.1:5000)

---

## Usage
1. Open the app in your browser.
2. Add and manage patient records.
3. Track basic health data.
4. Export reports as CSV (if implemented).
